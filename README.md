# adventofcode25

Solutions for [Advent of Code 2025](https://adventofcode.com/) challenges in Python.

## Project Structure

```
adventofcode25/
├── src/
│   └── day01/
│       ├── __init__.py
│       └── solution.py
├── tests/
│   ├── __init__.py
│   └── test_day01.py
├── requirements.txt
└── README.md
```

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running Tests

```bash
python -m pytest tests/ -v
```

## Running Solutions

```bash
cd src/day01
python solution.py
```
